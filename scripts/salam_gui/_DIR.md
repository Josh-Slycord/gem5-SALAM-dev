# SALAM GUI

PySide6-based graphical interface for gem5-SALAM simulation analysis.

## Subdirectories

| Directory | Description |
|-----------|-------------|
| dialogs/ | Dialog windows (connection, settings) |
| data/ | Data loading and parsing modules |
| resources/ | Static resources (icons, themes) |
| widgets/ | Custom PySide6 widget components |

## Files

| File | Description |
|------|-------------|
| __init__.py | Package initialization |
| __main__.py | Entry point for python -m execution |
| app.py | Application class and initialization |
| main_window.py | Main window implementation |
